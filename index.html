My Projects

Loan Finance

I developed a secure and user-friendly web application for managing loan approvals and disbursements using the Django framework. The application features a responsive interface built with HTML, CSS, and Bootstrap, and stores user data securely in a MySQL database. Users can submit loan applications, which are reviewed by the admin for accuracy before approval. After approval, users provide bank account details, and the admin manages loan disbursements efficiently. The system includes an EMI panel where users can pay monthly installments, with complete transaction history visible in both user and admin panels. Secure email verification and robust validation ensure safety and authenticity throughout the loan process. The project emphasizes seamless functionality, strong security, and an intuitive, user-friendly experience.
